# container4ctf

Docker container environment for CTF.

## base image

ubuntu official image. see: [ubuntu \- Docker Hub](https://hub.docker.com/_/ubuntu/)

## included specific packages

- `gdb`
- `tshark`
- `imagemagick`
- `peco`
- `nkf`

## usage

```bash
# build and run
$ npm run develop

# attach shell
$ npm run shell
```
